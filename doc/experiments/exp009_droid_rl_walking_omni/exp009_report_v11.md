# V11: vyコマンド範囲の最適化（±0.15 → ±0.20 m/s）

## 概要

V10をベースに、vyコマンド範囲を±0.15から±0.20 m/sに拡大する。V10ではvy範囲が狭すぎたことで横方向移動能力が喪失し（増幅率0.63x）、報酬コンポーネントが全面低下した。V9(±0.3)/V10(±0.15)/V11(±0.20)の3点データにより、vy範囲の応答曲線を取得し、横方向移動能力の回復とpenalty overkillの解消を狙う。

## 前バージョンからの改善事項

V10レポートの「次バージョンへの提案」推奨案Aに基づき、vyコマンド範囲を±0.20に設定。

| パラメータ | V10値 | V11値 | 変更理由 |
|-----------|-------|-------|---------|
| lin_vel_y_range | [-0.15, 0.15] | **[-0.20, 0.20]** | V10の±0.15は狭すぎ（増幅率0.63x）。V9(±0.3)とV10(±0.15)の中間で到達率70-80%域を狙う |

### V11でvy±0.20を選定した理由

1. **横方向移動能力の回復**: V10で増幅率0.63x（入力が減衰される状態）となり、横方向が実質的に死んでいた
2. **penalty overkillの解消**: vy範囲縮小によりtracking_lin_vel等の正報酬獲得機会が減少、penalty/positive比が実効的にoverkill域に移行していた
3. **3点応答曲線の取得**: V9(±0.3)/V10(±0.15)/V11(±0.20)のデータにより、vy範囲と各指標の関係を定量化できる
4. **人間の直観との一致**: 「0.2くらいでよかった」との所感に合致
5. **1変更1検証原則の遵守**: 報酬構成・action_rate scale・knee Kpは変更なし

## 考察と改善案

### 目視評価所見

目視評価において以下が観察された:
- 前後移動についてはびっこを引いた歩きが軽減されたが、まだ残存している
- 左右移動は幅を狭めたことで多少改善されたように見える

### 対話による多角的分析

以下、データ分析専門エージェント（DA）とロボティクス・物理学俯瞰エージェント（Lead）の対話形式で考察を行う。

---

**DA（Round 1: 初期定量分析）**:

V10→V11で最も顕著な変化5項目:

- **報酬全面向上**: 主要報酬が軒並み+40〜50%増（tracking_lin_vel 0.479→0.690 (+44%)、contact 0.285→0.413 (+45%)、single_foot_contact 0.184→0.264 (+43%)）。ペナルティも増加しているが報酬増が大幅に上回る。
- **Yawドリフト悪化（固定コマンド基準）**: drift rate 0.544→1.424°/s（**2.62倍悪化**）、Y偏位 -0.019→+0.574m。直進安定性が明確に低下。
- **左右非対称の構造変化**: L swing 0.280→0.318s（+14%延長）、R swing 0.269→0.261s（微短縮）。接地比もL50.8%:R44.4%で左寄り。非対称度は-15.1%→-14.3%でわずかに改善。
- **hip_roll振動の大幅改善**: vel RMS がL 0.66倍、R 0.50倍に減少。マイクロスタンス/マイクロスイングも完全消失（7.7%/15.4%→0.0%）。
- **横方向揺れの質的変化**: Roll std 3.27→3.81°に微増だが、Roll-横方向R²が0.029→0.142に上昇（増幅率0.63→1.51倍）。Rollが横方向運動により強く結合。

人間所感との対応:
- 「びっこ軽減だが残存」: 裏付けあり。非対称度-15.1%→-14.3%、hip_pitch相関-0.249→-0.289（交互歩行改善）。ただしL/R swing差が0.011s→0.057sに拡大しており、左足の滞留時間延長がびっこ感の残存要因と推定。
- 「左右幅改善」: 部分的裏付け。hip_roll振動の半減とマイクロ接触消失は改善を示すが、detrended横方向std 60.3→78.0mm（+29%）と横方向変位自体はむしろ増加。体感改善はhip_roll安定化による滑らかさ向上に起因する可能性。

Yaw信頼性チェック: ランダムコマンド評価のYaw drift=-4.39°はコマンド切替の影響を受け不適切。固定コマンド（FWD 0.3 0.0 0.0）データのみで算出: drift rate 0.544→1.424°/s、**Yawドリフトは2.62倍悪化**。

**Lead（Round 2: 物理解釈・仮説構築）**:

因果連鎖の特定:

1. **因果連鎖A（vy拡大→報酬回復→歩行品質全面向上）**: vy ±0.15→±0.20 → 横方向コマンドの到達可能域拡大 → tracking_lin_vel等の正報酬獲得機会回復 → penalty/positive比がoverkill域から健全域へ復帰 → 歩行品質指標の連動改善。
2. **因果連鎖B（hip_roll安定化の力学的メカニズム）**: vy±0.20で横方向コマンドが到達可能かつ無理のない範囲に入る → policyが安定したhip_roll戦略を獲得 → vel RMS半減、マイクロ接触消失。Roll周波数が0.38Hz→1.52Hz（歩行周波数1.50Hz）に同期化した事実は、V10の自励振動的な低周波動揺からV11の正常な左右体重移動パターンへの質的転換を意味する。
3. **因果連鎖C（振動エネルギー移転）**: hip_roll振動の半減と同時にR_hip_yaw vel RMS 1.52倍、L_knee_pitch vel RMS 1.34倍。振動エネルギーがhip_rollから他関節に移転した可能性があるが、exp007ほど深刻ではない。
4. **因果連鎖D（Yawドリフト悪化のメカニズム）**: vyコマンド範囲拡大 → 横方向移動戦略の獲得 → hip_yaw/hip_rollの左右差がYawモーメントを生成 → Yawドリフト増大。V9(±0.3)/V10(±0.15)/V11(±0.20)の3点データはvy範囲とYawドリフトの間に強い正の非線形相関を示す。

penalty/positive比は健全域に復帰と判定。

仮説:
1. Yawドリフトの主因はvy学習に伴うhip_yaw左右非対称の増幅
2. Roll周波数の歩行同期化はhip_roll制御の質的転換を示す
3. L_knee/R_ankle高周波増加がカスケード悪化の初期段階の可能性

**DA（Round 3: 仮説検証データ）**:

1. **仮説1（Yaw→hip_yaw非対称）: 強く裏付け。** L-R hip_yaw差: V10=+1.70°→V11=+8.33°（**4.9倍拡大**）。遊脚相でL+6.01° vs R-7.54°の系統的偏り。V11はL action mean=+0.25, R=-0.27で明確な非対称。
2. **仮説2（Roll歩行同期化）: 裏付け。** V11はL/R hip_rollとも1.50Hz単一ピーク集中、V10 Rにあった高調波（3.00Hz, 4.50Hz）がV11で消失。Roll-接地位相が対称的に同期（L lift-off: +2.98°, R lift-off: -4.46°→支持脚側にRollが傾く正常パターン）。
3. **仮説3（カスケード悪化）: 反証。** L_knee高周波は前半30.1%→後半26.0%で減少傾向。R_ankle横ばい（10.5→11.4%）。カスケード悪化兆候なし。

**Lead（Round 4: 統合考察・改善案）**:

Yawドリフト悪化の因果関係:
```
vy範囲拡大(±0.15→±0.20)
  → ポリシーがhip_yawを横移動の道具として積極利用
  → L/R hip_yawの系統的非対称が増幅(L-R差: +1.70°→+8.33°, 4.9倍)
  → 遊脚相での蹴り出し角度が左右非対称 → トルク非対称 → Yaw回転力発生
  → drift rate 2.62倍悪化 (0.544→1.424°/s)
```

vy範囲とYaw drift rateの関係は超線形（vy +33%でdrift +162%）。hip_yaw非対称の増幅がドリフトを支配。

びっこ感の質的変化: V10ではhip_pitch振幅差（蹴り出しの弱さ）が主因だったのに対し、V11ではswing duration差（左脚が長く浮いている）が主因に移行。「びっこが軽減されたがまだ残っている」との目視所感は、この振幅非対称→タイミング非対称への質的変化と整合する。

---

### 成功点

- **報酬全面向上**: tracking_lin_vel +44%（0.479→0.690）、contact +45%、single_foot_contact +43%。penalty/positive比がoverkill域から健全域に復帰
- **hip_roll振動の大幅改善**: vel RMS L 0.66倍、R 0.50倍。マイクロスタンス/マイクロスイング完全消失。Roll周波数が歩行周波数と同期化（0.38→1.52Hz）し、制御が質的に転換
- **バランスストロークの縮小**: L 9.0→5.3°（-40.7%）、R 8.1→5.3°（-35.1%）。hip_roll安定化により大きなバランス回復動作が不要に
- **カスケード悪化の不在**: L_knee/R_ankle高周波増加は時間的に安定しており、exp007型の連鎖的悪化兆候なし
- **hip_pitch交互歩行の改善**: 相関 -0.249→-0.289（ランダム）、固定FWDでは-0.340

### 課題

- **Yawドリフトの悪化（最重要課題）**: 固定コマンドFWD drift rate 0.544→1.424°/s（2.62倍悪化）。根本原因はhip_yaw L-R差の4.9倍拡大（+1.70°→+8.33°）。vy範囲とYawドリフトが超線形に結合しており、vy範囲微調整では根本解決にならない
- **びっこ感の残存**: hip_pitch非対称度は微改善（-15.1%→-14.3%）だが、swing duration差が0.011s→0.057sに拡大。振幅非対称→タイミング非対称に質的変化
- **横方向変位の増加**: detrended横方向std 60.3→78.0mm（+29%）。Roll-横方向増幅率が0.63→1.51倍に増大。ただしバランスストローク縮小により体感的には改善
- **base_heightペナルティ増加**: z mean 0.284→0.294m（+10mm）によりbase_heightペナルティが2倍化（-0.011→-0.022）

### 次バージョンへの提案

**案A（推奨）: hip_yaw対称性報酬の追加**

- **変更内容**: 左右hip_yawのアクション出力差にペナルティを課す報酬項を追加。`reward_hip_yaw_symmetry = -scale * (a_L_hip_yaw - a_R_hip_yaw)^2`
- **初期scale**: -0.15（hip_yawの2自由度のみに作用）
- **vy範囲**: ±0.20を維持（変更しない）
- **期待効果**: hip_yaw L-R差（現在+8.33°）の直接的抑制、Yaw drift rate の大幅低減（目標 < 0.8°/s）
- **リスク**: scaleが過大だと横移動能力が制限される（hip_yawは横移動にも必要な自由度）。対称性制約が強すぎると「両脚ともゼロ」解に収束し横移動能力が喪失するリスク
- **根拠**: V11のYawドリフトの主因がhip_yaw非対称であることは仮説検証で強く裏付けられている（L-R差4.9倍拡大、遊脚相での系統的偏り）。因果関係の最上流を直接ターゲットする案であり、報酬ベースのソフト制約で副作用も制御しやすい

**案B: 速度二乗EMA対称性報酬（V12として既に準備済み）**

- **変更内容**: 左右の速度二乗EMA平滑化値の差にペナルティ（symmetry_vel_ema scale=0.3）
- **期待効果**: 左右の運動エネルギー均等化→swing duration非対称の間接的改善、びっこの軽減
- **リスク**: Yawドリフトへの効果は間接的（hip_yaw非対称を直接制約しない）
- **根拠**: 既に準備されており実装コスト低。ただしYawドリフトの根本原因への直接効果は限定的

**案C: vy範囲の微調整（±0.20 → ±0.175）**

- **期待効果**: Yaw drift rateの低減（~0.9°/s程度を予測）
- **リスク**: 問題の先送り。横方向移動能力の再低下。V9/V10/V11で既に3点データを取得済みであり微調整の情報利得は低い

## まとめ

V11（vy ±0.20）はV10で生じた「penalty overkill→歩行品質全面低下」を正しく解消し、tracking_lin_vel +44%、hip_roll振動大幅改善、バランスストローク -35〜-41%、カスケード悪化の不在という複数の成功を同時に達成した。人間の目視でも「左右の幅が狭まった」「びっこが軽減された」と確認された。一方で、vy範囲拡大の代償としてhip_yaw非対称が4.9倍に拡大し、Yawドリフトが2.62倍悪化した（固定FWD drift rate 0.544→1.424°/s）。これはvy範囲とhip_yaw自由度の利用度が超線形に結合していることを示しており、vy範囲の微調整では根本解決にならない。次のステップとして、hip_yaw対称性報酬の追加（案A）によりYawドリフトの根本原因を直接制約しつつ、V11で獲得した歩行品質改善を維持することを推奨する。
