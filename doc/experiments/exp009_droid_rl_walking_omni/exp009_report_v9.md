# V9: knee Kp=50（単独、関節グループ別Kp/Kd再検討の第一歩）

## 概要

V8をベースに、knee_pitch Kp 35.0→50.0（単独変更）を実施する。V8でgait_frequency 1.5Hzによる交互歩行回復が達成されたことで、V7レポートで計画した「gait_frequency確立後のKp/Kd再検討」の条件が成立した。V7ではknee Kp=50 + ankle Kp=20の同時変更で効果分離が不可能だったため、V9ではknee Kp=50を単独で検証し、1.5Hz歩行環境における膝応答性の強化効果を評価する。

## 前バージョンからの改善事項

V8レポートの「次バージョンへの提案」に基づき、knee Kp=50を単独で実施。

| パラメータ | V8値 | V9値 | 変更理由 |
|-----------|------|------|---------|
| knee_pitch Kp | 35.0（デフォルト） | **50.0（kp_overrides）** | V7計画のKp/Kd再検討第一歩。膝は最大負荷関節。swing中の制御力増強で正報酬回復を狙う |

### V9でknee Kp=50を単独変更する理由

1. **V7の教訓**: knee Kp=50 + ankle Kp=20の同時変更で効果分離不可。1変更1検証原則に基づき単独検証
2. **物理的合理性**: 膝は最大負荷関節であり、Unitree G1でもhipの1.5倍のKpを使用
3. **V8の正報酬低下への対策**: swing_duration -59.2%、step_length -28.4%を膝制御力増強で改善しうる
4. **リスク低減**: ankle Kp減少とは異なり、制御帯域低下・接地バウンス・報酬欺瞞のリスクがない
5. **高速歩行環境への適応**: gait_frequency=1.5Hzでは片足支持時間が0.667sと短く、関節の応答性がより重要

## 考察と改善案

### 目視評価所見

目視評価において以下が観察された:
- 安定感は多少増加した
- 前進と後退の動きはまだ悪いまま
- 指令値0でも静止できていない（不必要な足上げが継続）
- 前後移動でびっこを引いた歩きになっている（後退自体はちゃんと後ろに進む）
- 左右方向の歩幅が大きすぎる（もっと狭めて可愛くしたい）
- 左右方向は可動域の制約上、目標速度を下げることが妥当と考えられる（速度到達域の楕円化イメージ）

### 対話による多角的分析

以下、データ分析専門エージェント（DA）とロボティクス・物理学俯瞰エージェント（Lead）の対話形式で考察を行う。

**注意**: 初回分析ではランダムコマンド評価のみを根拠としたため、「後退指令で前進する」等の誤った結論に至った。方向固定評価（20秒、seed 1）をV9/V8双方で追加実施し、以下は修正後の分析である。

---

**DA（Round 1: 初期定量分析 — 修正版）**:

V8→V9で最も顕著な変化を5項目特定した:

1. **方向固定時のYawドリフト激悪化（最重要発見）**: ランダム評価ではYaw 82%改善（2.011°/s→0.357°/s）を示したが、方向固定20秒評価では**V9が5〜7倍悪化**。前進: +60.33°(V9) vs +11.52°(V8)、後退: -75.95°(V9) vs -10.40°(V8)。ランダム評価の4秒コマンド切替が方向依存Yawドリフトを統計的に相殺し、問題を隠蔽していた。
2. **右脚マイクロコンタクト激増**: Rマイクロスタンス 14.3%→35.3% (2.47x)、Rマイクロスイング 14.3%→33.3% (2.33x)。右脚の接地制御が不安定化。
3. **全関節DOF速度RMS 1.28-1.88x増加**: 全関節例外なく悪化。特にhip_yaw L=1.88x。アクション変化率RMSも全体+42%。
4. **右脚高周波振動の異常増大**: R_knee_pitch高周波比率 7.4%→22.8% (3.08x)、R_hip_pitch 9.1%→21.2% (2.33x)。右脚に局所的な振動問題が集中。
5. **後退は機能するが品質低下**: 後退追従率74.7%（V8: 78.3%）で後退自体は成立。ただし横方向ドリフト3.011m（V8: 0.575m）、Yawドリフト-75.95°（V8: -10.40°）と軌道品質は大幅劣化。

所感との対応: 「安定感多少増」は横方向増幅率2.68x→1.71x改善で裏付け。「びっこ」はknee非対称度+16.5%・右脚マイクロコンタクト2.47x・右脚振動集中で強く裏付け。「後退が不自然だが進む」は追従率74.7%+Yawドリフト-75.95°で定量的に裏付け。「静止不可」は空中相5.0%(V8:1.2%)で裏付け。

注目トレードオフ: (1) ランダム評価の短期Yaw対称性改善⇔方向固定評価での累積Yawドリフト激悪化、(2) 速度追従改善⇔振動悪化（action_rate penalty抑制力不足）。

**Lead（Round 2: 物理解釈・仮説構築）**:

**構造的右バイアス仮説**: knee Kp 35→50 (+43%) がL/R非対称性を増幅し、「構造的右バイアス」を生成するメカニズムを以下のように推定する。

V8時点でR hip_pitch rangeがLの1.35倍と既に右脚が大きく振れる傾向があった。Kp増加により、V9ではR knee range = L x 0.72（膝が狭い）、R ankle = L x 1.36（足首で補償）となった。重要なのは、**R膝のswing-phase action非対称性が方向に依存せず一貫してR>L**であること（前進+29.7%、後退+52.1%）。V8では方向に応じて非対称性が反転していたが、V9ではKp増加により右膝が方向によらず一貫して支配的になった。

この「構造的右バイアス」は:
- ランダム評価（4秒切替）では短いため累積せず、むしろ各ステップの対称性向上として観測
- 方向固定評価（20秒連続）では累積し、持続的なYawモーメントとして顕在化
- 前進では+60°（右旋回）、後退では-76°（左旋回）と、**符号が反転**する（推進力方向とYawモーメント方向の幾何学的関係から予測通り）

penalty/positive ratio判定: V9の正報酬合計~2.45、ペナルティ合計~-0.15で ratio~0.061。exp007危険域(>0.10)からは十分遠く健全域内。ただしang_vel_xy 70%悪化・orientation 76%悪化は振動系ペナルティの急増を示す。

**DA（Round 3: 仮説検証データ）**:

Lead要求への回答:

1. **方向固定評価によるYawドリフトの定量比較**:
   - 前進20s: V9 +60.33° vs V8 +11.52°（5.24倍悪化）
   - 後退20s: V9 -75.95° vs V8 -10.40°（7.30倍悪化）
   - 符号反転パターン（前進→右旋回、後退→左旋回）はLead仮説の「構造的右バイアス」と整合
2. **L/R別関節ROM**: V9でR knee_pitch range = L x 0.72（膝が狭い）、R ankle_pitch = L x 1.36（足首で補償）。**V9右脚は「膝の高Kpで振り抜き抑制→足首逃げ」パターン**を示す。
3. **swing-phase knee action非対称性**: V9はR>Lが方向非依存（前進+29.7%、後退+52.1%）。V8は方向依存的に反転。Kp増加が方向非依存の構造的バイアスを固定化した証拠。
4. **action_rate実装**: L2^2（二乗和）、scale=-0.01。V8→V9でL2^2生値 1.614→3.281 (2.03x)。scale不足は成立するが、Yawドリフトの根本原因はaction_rateではなくL/R非対称性。
5. **vy到達可能速度**: V9実効到達はvy正方向~0.16 m/s、vy負方向~0.10 m/s。現行コマンド上限±0.3 m/sは到達率最大53%であり、到達不能領域が大きい。

**Lead（Round 4: 統合考察・改善案）**:

**V9の本質的問題: ランダム評価では検出不能な方向依存Yawドリフト**

本分析の最も重要な発見は、ランダムコマンド評価と方向固定評価の乖離である。ランダム評価は4秒ごとにコマンドを切り替えるため、方向依存的なYawドリフトが切替ごとに相殺され、あたかも改善されたように見える。しかし実際には、knee Kp増加がL/R非対称性を「構造的右バイアス」として固定化し、一方向への持続歩行で5〜7倍のYawドリフト悪化を引き起こしている。

**悪化メカニズムの因果関係（修正版）**:
```
knee_pitch Kp 35→50 (+43%)
  → ポリシー過剰入力 (action_rate L2^2 2.03x)
    → R膝のswing-phase action非対称性が方向非依存に固定化
      → R knee range = L x 0.72 (ロックアップ) + R ankle = L x 1.36 (逃げ)
        → 持続的な右方向Yawモーメント
          → 前進: +60° / 後退: -76° の累積Yawドリフト（V8比5-7倍悪化）
          → 横方向ドリフト: FWD +2.4m, BWD +3.0m（V8比5-17倍悪化）
  → ランダム評価では4s切替が相殺効果 → 見かけ上82%改善（誤検出）
```

**成功点**: (1) 速度追従+24%（膝剛性増による推進力伝達効率向上）、(2) ランダム評価での短期Yaw対称性向上（切替ごとのステップ精度は実際に向上）、(3) 横方向増幅率36%改善（2.68x→1.71x）、(4) L stance hip_roll std 38%減（剛性向上による安定感向上、「安定感多少増」の所感と整合）、(5) 収束安定性改善（Last 50 std 0.303→0.122）。

**改善案の学術的再考察**（既存V9レポート案に囚われず中立的に検討）:

| 案 | 内容 | 期待効果 | リスク | 1変更1検証 |
|----|------|---------|--------|-----------|
| A | vyコマンド範囲 ±0.3→±0.15 | 左右歩幅縮小、楕円化、到達不能コマンド除去 | Yawドリフト・びっこの根本解決にはならない | 遵守 |
| B | knee Kp=50→42 (中間値) | 構造的右バイアス緩和+一部のKp効果維持 | 変化量小で効果不明確 | 遵守 |
| C | action_rate scale -0.01→-0.02 | 過剰入力抑制、振動間接改善 | 速度追従低下、根本原因（L/R非対称）への対処ではない | 遵守 |
| D | knee Kp revert to 35 | 構造的右バイアス完全除去 | V9成功点喪失（速度追従+24%、増幅率改善等） | 遵守 |
| E | vy ±0.15 + Kp=42 同時 | A+Bの同時獲得 | 1変更1検証原則違反 | 違反 |

**推奨案: A（vyコマンド範囲の楕円化 ±0.3→±0.15）** を選定。理由: (1) 人間の要望（左右歩幅縮小・楕円化）への直接対応、(2) DAデータが示すvy実効到達速度(~0.16 m/s)とほぼ一致する範囲設定であり物理的妥当性が最も高い、(3) 到達不能コマンド除去による訓練効率向上が前後方向にも波及しうる、(4) 1変更1検証原則を完全遵守、(5) vy楕円化でびっこが改善しなければ、構造的右バイアスが主因と確定でき、次にKp=42を検証する根拠となる。

**フォールバック**: vy楕円化後もびっこ・Yawドリフトが残存する場合、V11でknee Kp=42に変更。V9のKp=50で得られた知見（Yaw短期対称性改善の代償としてL/R構造的バイアス固定化）を踏まえた中間値探索となる。

### 成功点

- **速度追従報酬 +24%**: knee Kp増による推進力伝達効率向上。tracking_lin_vel 0.768→0.952
- **ランダム評価での短期Yaw対称性向上**: 4秒切替ごとのステップ精度は向上（ランダム評価でYaw 82%改善として観測）
- **横方向増幅率 36%改善**: 2.68x→1.71x。V9の成功指標（<2.0x）を達成
- **バランスストローク大幅増加**: L+128%, R+53%。hip_rollを活用した能動バランスの学習が進展
- **収束安定性**: Last 50 std 0.303→0.122で収束が改善
- **後退歩行の成立**: 追従率74.7%で後退自体は機能する（V8: 78.3%と同等水準）

### 課題

- **方向固定Yawドリフト激悪化（最重要）**: 前進+60.33°（V8比5.2倍）、後退-75.95°（V8比7.3倍）。knee Kp増加による「構造的右バイアス」が原因。ランダム評価の82%改善は4秒切替の相殺効果による誤検出
- **横方向ドリフト激悪化**: 前進+2.4m（V8比16.6倍）、後退+3.0m（V8比5.2倍）。Yawドリフトに伴う軌道逸脱
- **「びっこ」歩行**: R knee range = L x 0.72の非対称性。R膝ロックアップ→足首逃げパターン。swing-phase knee action非対称性が方向非依存にR>L固定化
- **右脚カスケード不安定**: R膝高周波振動3.08x、Rマイクロコンタクト2.47x
- **全関節振動増大**: DOF速度RMS全関節1.28-1.88x増加、アクション変化率+42%
- **Roll std悪化**: +44.9%（3.09°→4.47°）。周期性も低下（0.769→0.478）
- **静止不能**: 低速コマンドでも空中相4%(V8:1%)

## まとめ

exp009 V9はV8ベース + knee_pitch Kp 35.0→50.0（単独変更）を実施した。速度追従+24%・横方向増幅率36%改善・収束安定性向上という成果を得た一方、方向固定評価により**ランダム評価では検出不能な重大問題**が発覚した。ランダム評価ではYawドリフト82%改善を示したが、方向固定20秒評価ではV9のYawドリフトがV8比5〜7倍悪化（前進+60°、後退-76°）。knee Kp増加がL/R非対称性を「構造的右バイアス」として固定化し、一方向への持続歩行で累積Yawドリフトと横方向ドリフト（V8比5〜17倍）を引き起こしている。後退歩行自体は追従率74.7%で成立するが、不自然な動きとYawドリフトが課題。V10ではvyコマンド範囲の楕円化（±0.3→±0.15 m/s）により、到達不能コマンドの除去と左右歩幅の適正化を検証し、改善しなければV11でKp=42にフォールバックする。
